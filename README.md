# 📝 Lista de Tarefas

Uma aplicação simples de **Lista de Tarefas (To-Do List)** desenvolvida com **HTML, CSS e JavaScript**.  
O projeto permite adicionar, remover e salvar tarefas diretamente no navegador utilizando **LocalStorage**, garantindo que as tarefas permaneçam salvas mesmo após atualizar a página.

---

## 🚀 Funcionalidades

- ➕ Adicionar novas tarefas
- ⌨️ Adicionar tarefas pressionando **Enter**
- ❌ Remover tarefas com confirmação
- 💾 Salvamento automático no **LocalStorage**
- 🔄 Recuperação automática das tarefas ao recarregar a página
- 📱 Interface simples e fácil de usar

---

## 🖥️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **LocalStorage API**
